## 内容
大家好我是唐斩
ClaudeCode在11月1日的时候正式更新了原生安装程序
之前需要先安装nodejs，再通过npm安装 ClaudeCode了。

安装方法
- mac 上使用 homebrew，brew install claude-code
- mac、linux、WSL(windows 上的原生 linux 环境)，curl -fsSL https://claude.ai/install.sh | bash
- windows power shell下，irm https://claude.ai/install.ps1 | iex

如果是已经安装了ClaudeCode的，可以使用 claude install 命令，从 nodejs 切换到原生安装程序，获得更好的体验。记得关闭所有运行中的 claude code 实例后再执行。
