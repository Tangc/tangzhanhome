多智能体，就是让多个专用智能体，像团队一样互相协作，完成更复杂的任务
评估，就是通过量化来衡量智能体的效果，以实现改进。

想学 Agent 开发的朋友们，看这一个课程就够了。
完全免费，视频学习，附带代码练习。

今天看完了吴恩达在DeepLearning上的 Agentic AI 课程。
内容非常适合做Agent开发的新受学习。

Agentic 相比 Agent 的意思是系统不是非黑即白的Agent或者非Agent，而是根据实际情况适当 Agentic 化的具备AI代理化能力的系统。

课程围绕Agent的四种设计模式展开，分别是
- Reflection 反思，通过增加外部输入(可以是搜索，也可以是用户输入)，让LLM的回答效果显著提升。相当于认知迭代法，加强LLM的思考深度。

- Tool Use 工具使用，从 LLM 内自带的 Function Call，到 Agent 定义的 Tools (一个函数，读取数据库等，一次搜索等)，到 MCP 协议下的Tool调用(把 MCP 装入 Tool 里，给Agent开挂，链接整个互联网世界)。极大扩展 LLM 的能力和触达范围。

- Planning 规划，通过拆分使得复杂任务的执行更可控，由 LLM 负责规划，能极大泛化Agent能力，不受开发者经验的束缚。

- Multi-Agent 多代理，协调多个专门的Agent来处理任务的不同部分，通常两种协调方式，中心化协调，和去中心化协调，根据实际情况选择。

课程全程是视频讲解，总共3个小时，一下午就能看完。还有配套的代码课程，用的是python语言，跟着教程走就可以复现课程中的 Agent 功能，非常适合新人上手，做过案例了再迁移到自己想做的系统中。
课程是全英文讲解，可以开启沉浸式翻译的动态字幕翻译功能，就可以看到实时双语弹幕了。
