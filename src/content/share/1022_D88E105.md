## 1、大模型投资竞技场
国外有好事之徒，把世面上的6个顶尖大模型拉到一起，给每个分配1w刀，限定6种数字货币，然后让他们投资。
这个项目叫做 Alpha Arena by Nof1 声称的目的是，测试大模型的投资能力。

6种大模型是，GPT-5，Claude-Sonnet-4.5，Gemini-2.5-pro，Grok4，DeepSeek-v3.1，Qwen3-MAX。
6种数字货币是，BTC、ETH、SOL、BNB、SOGE、XRP。

从项目的公开网站可以看到很清晰的 battle 信息，包括
- 实时收益和历史收益曲线。
- 6种数字货币的实时价格。
- 6种大模型的历史收益曲线，实时资产数(从1w刀开始)。
- 6种大模型的历史交易情况(包括买、卖)，聊天记录(接收的提示词和反馈)。
第一赛季会到11月3日结束。
现在的情况是，只有咱们的两款国产开源大模型，deepseek和qwen的收益是正的，并且跑过了全买入和持有BTC的基准线(YYDS)。
什么国外的顶尖大模型(gpt和gemini)，底裤都要赔光了，来搞笑的。网友们都调侃，幻方不愧是做量化的，DeepSeek搞投资是要厉害点。

从这个事情的思考：
- 投资是很专业的事情，不可等闲视之。AI来了也是多数赔钱，少数微赚。
- 这个实验有价值，但也有局限，比如，AI的能力运用和提示词有很大关系、AI们得到的只是定时输入的有限信息，没有实时的世界信息等。形式可以作一个参考，但结果不能太认真。也期待实验结束后组织方给出的报告，看看有没有值得注意的内容。
- 让AI参与现实世界的可衡量的活动，也许真是一种评估的思路。
- 下场在同等条件下挑战AI，你敢不敢试试？

## 2、继续谈 Claude Agent Skills
Claude Agent Skills 也许是 Claude 的 AppStore 时刻。众所周知是 AppStore 完成了 iPhone 的商业应用生态，让 iPhone 走上神坛。
Skill能力之于 Claude ，完全可类比智能手机APP之于手机操作系统。而前段时间推出的 plugin market，就正是对应 AppStore。
Skill 让 Claude 的普通用户可以自定义“能解决具体问题的技能”，注册到 Claude 中使用，并且 Skill 可以通过 plugin market 发布。
让用户可以对 Claude 智能体的使用，达到 write prompts once，run everywhere 的效果。

但和 AppStore 不同的地方在于， Claude 的产品保持了自己的 Linux 哲学，简洁、透明、可组合、开放和自由。
Skill，plugin 都是基于文本文件的，Skill是聚焦单一能力，但可组合使用的。
连 plugin 的 market 都是可以自定义公有源或私有源的。
可以说，未来不会出现套壳的Claude，因为通过开放能力就可以任意重塑成自己需要的模样。
