## 中间内容 (15s注意力，路标预告)
我来助你，用这个github上开源的免费工具 grec,
一行命令就能用AI把整个仓库的 commit message 改写成规范的有意义的内容。
两个步骤搞定

步骤1，配置环境
以mac为例
先安装 nodejs

然后配置大语言模型
可以选本地的 ollama 或者远程的 openai
git commit是短文本，
推荐用ollama调用本地部署的小模型，
效果足够，数据安全，零成本

先homebrew安装 ollama，然后下载 gemma3 模型
更多模型可以在ollama官网选择

用 ollama server 启动服务
用 ollama run xx 命令测试效果

环境就准备好了

步骤2，重写commit
来到项目文件夹
执行一行命令，重写最近10条commit

npx grec --max-commits 10 \
  --provider ollama \
  --model gemma3 \
  --no-skip-well-formed \
  --template "[branch] type: message" \
  --language zh-cn
  

用我的切图项目为例，执行效果

改完规范多了。

grec 也可以在 commit 的时候自动生成 message
需要执行这个命令安装hook
然后添加这几个配置项
