# 大家好，我是唐斩，这是第97天第114篇分享。

## 1、ClaudeCode 支持原生安装了
ClaudeCode 终于支持原生安装了，mac上安装不再需要依赖 nodejs 了，自动更新更加稳定(我遇到过无数次自动更新失败，需要手动删除文件夹的情况)，安装程序可独立执行。
对于非程序员群体，安装 nodejs 多一道门槛多一些失败的可能性，现在终于可以直接安装了。
安装方法
- mac 上使用 homebrew，brew install claude-code
- mac、linux、WSL(windows 上的原生 linux 环境)
- windows power shell下，irm https://claude.ai/install.ps1 | iex

如果是已经安装了ClaudeCode的，可以使用 claude install 命令，从 nodejs 切换到原生安装程序，获得更好的体验。记得关闭所有运行中的 claude code 实例后再执行。


参考内容

- https://www.reddit.com/r/ClaudeAI/comments/1ol0af7/claude_codes_native_installer_is_now_generally/


## 2、hackernews 上对 openai atlas 的讨论
openai 的 atlas 发布后，hackernews 上一个帖子对它展开了长篇讨论，提取几个有意思的观点
- 观点1，AI浏览器是新的信息垄断。话题发起者写了一篇blog，表示AI浏览器是违背Web开放精神的，Atlas会把AI产生的回答内容伪装成网页形式返回，并不是真实网页，而用户识别不出来。会产生信息的封闭和垄断。就像曾经担心谷歌对搜索结果的干预垄断一样。
- 观点2，隐私与监控焦虑。担心AI浏览器会提取用户行为用于模型训练，不仅仅是浏览网页的行为，包括在浏览器里的所有行为。甚至有人因此搭建了一个自用的LLM，联想到youtube的游戏圈top1博主也在自己搭建一个10卡集群的LLM。（有点末日地下堡的味道了）
- 观点3，AI浏览器和Google搜索各有场景，比如Google搜索的结果直观，而AI浏览器在需求模糊时表现的更好。符合我们的使用直觉，ChatGPT有报告说明教育和学习就是当下和AI聊天的一大场景。
