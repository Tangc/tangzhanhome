# 大家好，我是唐斩，这是第112天第129篇分享。


## 使用 Gemini 的 n 种方法

1. Gemini 官网
https://gemini.google.com
右下角选择 思考(3 Pro) ，然后对话。
注意有每日限额。

2. Google AI Studio
https://aistudio.google.com/prompts/new_chat
选择 Gemini 3 Pro Preview，然后在下方对话。
AI Studio 本身是用来体验 API 效果的，既然能免费用，就不客气了。


3. Google Gemini API
https://ai.google.dev/gemini-api/docs/gemini-3
调用 谷歌 的API服务使用 Gemini。
和 Cherry Studio 搭配，或者自己的应用里对接上 Gemini 3 Pro。
虽然 API 是付费使用，但开通 Cloud 账号会有90天的赠额可以免费使用。

阶梯定价
2 美元 / 12 美元（<20 万个 token）
4 美元 / 18 美元（>20 万个 token）

能用一阵了。

4. Google Search AI模式
https://www.google.com/
谷歌搜索的首页，选择 “AI模式”，再AI Mode选择 3 Pro。
Gemini 付费用户才能选择 3 Pro，不能免费使用。
这里的搜索记录和Gemini是不互通的。


5. Antigravity + Gemini 3 Pro
https://antigravity.google/
谷歌推出的 AI编程 IDE，对标 Cursor，Windsurf。
它其实就是 Windsurf 的硬分叉谷歌版本(付了买断版权费的)。
现在测试阶段， Gemini 3 Pro 免费使用，但是额度不多。
用完了可以切换到 Claude Sonnet 4.5 免费使用。
卡登录问题看我的视频有介绍。


6. Cursor + Gemini 3 Pro
如果是 Cursor 的订阅用户，可以直接切换到 Gemini 3 Pro 模型使用。
