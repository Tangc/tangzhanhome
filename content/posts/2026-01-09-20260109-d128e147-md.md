# ClaudeCode 的开源挑战者，AI编程最佳实践的集大成者 OpenCode


## OpenCode 简介
不登录，不绑卡，没有网络问题，国内顶尖模型免费用打开终端输入命令，敲下回车立刻就能开始生成代码，这就是能力比肩 cloud code 同时完全开源，完全免费，完全开放的 AI 编程界新星 open code 。
Github地址 https://github.com/anomalyco/opencode

## 一键安装
mac、windows、linux 全平台支持！安装使用超级简单，全部只需要这一行命令就可以完成 
curl -fsSL https://opencode.ai/install | bash

mac 下还可以用 home brew 安装
brew install opencode

我推荐geek用 bun 安装。
bun install -g opencode-ai
这个 bun，就是刚被 ClaudeCode 的公司 Anthropic 收购的最新最强的前端工具生态ALL IN ONE产品，opencode能这么快产出多端APP，和bun的跨平台能力二进制包能力有非常大的关系。bun这条线非常值得关注。

## 何以牛逼
opencode这个研发团队非常有意思，我完全想不到 TUI(terminal ui)的体验可以优秀到这个程度，追查了一下这个团队就是以TUI闻名，曾经做过TUI的电商产品，terminal 里购物？不敢想。

opencode的3大牛逼之处。完全开源免费不绑定模型厂商，AI编程工具最佳实践的集大成者，繁荣的社区插件

### 完全开源免费不绑定模型厂商
工具本体完全开源，免费使用，最宽松的 MIT 协议！开源意味着透明，完全符合技术人追求自由追求可控的精神，深得开源社区喜爱，已经有5.6w star了，还在急速增加中。
开源意味着，任何不满意的地方，我完全可以自己二创，或者贡献回主分支或者就用自己的分支，有开发能力的企业也可以基于社区版本开发自己的企业版。
另一面就是 ClaudeCode 的闭源，只能通过体感判断他强，通过逆向工程窥探他的设计，本体的能力迭代依赖于Anthropic团队，不可控，受制于人，不可二创改造。
不绑定模型厂商允许连接任意大模型，仅仅是推荐用 opencode 团队开发的 zen (类似 openrouter 的聚合器)，完全可以利用所有已经订阅过的产品，甚至可以集中起来一起同时用，Claude、GPT、Gemini 三花聚顶，你就说爽不爽吧！

### AI编程工具最佳实践的集大成者
更加劲爆的是 open code 完美支持！Claude 的 agent 直接把 MD 文档复制过来，立刻就可以使用。完美支持 skills、MCP等，甚至文件都不用动，他兼容去读 claude 的配置文件。模型切换的地方能看到 claudecoderouter的影子。全形态可用，CLI、web、desktop、插件。 
当前AI编程工具必备的能力聚合了个遍，现在用上 opencode，就是把最佳实践开箱即用起来，这还有什么不用的理由吗？

### 繁荣的社区插件
最最无敌的是社区， oh-my-opencode 插件
https://github.com/code-yeongyu/oh-my-opencode

提供了开箱就可以使用的赛博研发军团，什么ClaudeCode的Agent、Skill，不用折腾了，用这个插件就几乎是最顶级的配置，用上最强大模型三花聚顶。没有订阅也可以用其他的代替。
其实赛博军团模式，对不那么顶尖的模型提升更大，类似于通过外置 COT ，提升模型思考能力。
