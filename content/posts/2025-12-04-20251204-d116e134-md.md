# 大家好，我是唐斩，这是第116天第134篇分享。

## claudecode的最佳配置工具 zcf
编程最大的劝退点不是写代码，是永远不知道哪里不对的环境配置。
AI编程也是，ClaudeCode 更是。

发现一款零配置的环境配置工具 zcf。
不用写一行配置，就可以把 ClaudeCode 安装和配置好，全平台支持。
https://github.com/UfoMiao/zcf

以mac为例，从零开局，两步完成

### 步骤1、安装nodejs
brew install node

设置别名
alias zcf="npx zcf"

### 步骤2、通过 zcf 引导安装 claude 并完成配置
执行 zcf 命令
- 语言，显示语言和AI返回语言
- 安装 ClaudeCode，如果已经安装过的话，就是更新版本。（这里有一个bug，zcf只能管理node安装的claudecode，如果是原生安装的，它管理不了)
- 设置API配置
 - 用官方，登录claude官方账号（网络要求高，封禁风险大
 - 自定义，添加配置
   - 用模板，集合了 302ai和packycode，国产厂商 glm、minimax、kimi(注意不是kimi for coding)
   - 自定义，需要 url，token，模型名（kimiforcoding 用这个方式
 - 用 CCR。后续分享，优点在于动态调度(但要求必须配置模型名称，我有不能配名称的情况，就用不上😂)
- 工作流、输出风格、MCP，全选安装就行

完成，没有写一行配置，这就可以开始冲浪了！
