# 大家好，我是唐斩，这是第77天第93篇分享。

## 1、VibeCoding 50技巧(5) ClaudeCode 的 VSCode 插件
如果你VibeCoding时用的是命令行Agent， 比如 ClaudeCode Cli，会遇到和Agent交互的同时，在命令行不方便查看一些文件内容，或者同时做一些git操作。还要单独开一个terminal的问题。
我的解决办法是使用 ClaudeCode 的 VSCode 插件。
既能方便查看文件内容，使用所有VSCode的插件，也能完美运行 ClaudeCode，极度舒适。
最近 ClaudeCode 推出 sonnet4.5 的同时，升级了Cli和插件，更新了UI，也支持点击交互，IDE感更强了。
这么搭配，VSCode 从IDE瞬间变身原生 Code Agent。(IDEA中也有ClaudeCode插件可以使用)

因为一个窗口可以打开多个CodeAgent，可以有更丰富的玩法
- 多Agent开发团队，比如 Codex 偏稳重，ClaudeCode 更雷厉风行，那么久让 Codex 来出方案，让 ClaudeCode 来负责执行。
- 多Agent大逃杀，比如让多个Agent轮流开发和Review代码，互相严厉Review，卷出最合理的方案和最精炼的代码。
- 驱虎吞狼，可以让一个Agent，驱动另一Agent，一直干，独立工作更长的时间。(见过分享的案例)

OpenAI 刚推出正式版的CodeX，以及 VSCode 自带的 Copilot 也都支持插件模式。兜兜转转啊，最开始的 copilot、cline等都是插件形式，然后 Curosr、Windsurf 的套壳IDE模式，接着是 ClaudeCode 和 gemini 的 cli 模式，到现在再次回到插件中，但和IDE集成度更高。

插件在 VSCode 的侧边栏 Extensions 里，搜索就可以下载使用

插件地址
https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code
https://marketplace.visualstudio.com/items?itemName=openai.chatgpt



## 2、VibeCoding 50技巧(6) 国服福音 GLM4.6
国服VibeCoding人的福音
如果你因为网络问题、账号问题、封号问题、费用问题，还没用上 VibeCoding 的话，完全可以用智谱GLM4.6体验一下，可以无缝接入ClaudeCode客户端。
智谱是国内唯一支持按月付费的顶级编程大模型，对开发者非常友好，首月最低甚至只要20元，我最开始是把它当claude的兜底，最近claude又反华又降智还封号，我已经切过来一段时间了，我项目的复杂度GLM4.6完全够用。
KimiK2我其实也用了一段时间，但API付费太贵了，因此我还养成了 compact 的好习惯(T_T不压缩上下文简直用不起)。
这个月费肯定是不贵的，从智谱官网看，他是要走国外 Anthropic 的路子，从编程而不是从大而全的模型来切入市场，有一个可能的思考是，编程之于未来的赛博世界的软件，就像人类的手之于物理世界的工具。比特世界需要编程来生成工具，正如物理世界人类要用手来制造工具，工具之于人是人类迅速发展的原因之一。所以做好编程模型不是偏安一隅，可能是正中靶心的。
实际使用的体验，GLM4.6 的水平至少和 sonnet4 差不多，上下文也达到了200K，对复杂项目也非常友好，从官网发布的评测看，也是差不多的水平，并且很有信心说是国内最好的水平，而且可以无缝接入 ClaudeCode 客户端，非常值得一试。
注意智谱的国内域名是 bigmodel.cn


## 3、蚂蚁的大模型
蚂蚁发布了自己的1T参数的非思考模型，提供API接入。
值得关注的点是，通过优化训练方式提高了模型的“效率”，“在有限的输出下”达到了顶尖水平。
官方介绍说后面还有推理模型和多模态模型。
